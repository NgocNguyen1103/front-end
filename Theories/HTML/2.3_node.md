# Self closing tag

**General:** a element that does not need ``<\..>`` to close (not like: ``<p> <\p>``)

**Symbol:** ``<.. \>``
````
<hr \>: horizontal
<br \>: Break
````

## Example
````
<h1> Web title <\h1>
<\hr>
<h2> Application 1 <\h2>
<h2> Application 2 <\h2>
````
**Result:**

**Web tille**
---------------------------------------
Application 1

Application 2
